URL Shortener Microservice

My implementation of freeCodeCamp's URL Shortener API project, created as part of the Chingu FCC Speedrun Challenge

User Stories

  1. I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  2. If I pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.
  3. When I visit that shortened URL, it will redirect me to my original link.

Usage Examples

Creating a new "short" URL:

http://www.example.com/new/https://www.freecodecamp.com

Sample JSON response:

{"original_url":"https://www.freecodecamp.com","short_url":"http://www.example.com/6"}

Using the "short" URL:

http://www.example.com/6 => redirects to => https://www.freecodecamp.com
